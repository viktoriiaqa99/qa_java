# Feline Unit Tests (Sprint 6)

Проект модульного тестирования учебного приложения для исследования семейства кошачьих.
Работа выполнена в рамках финального проекта 6 спринта.

Цель проекта — покрыть unit-тестами классы бизнес-логики и обеспечить полное покрытие кода с использованием современных инструментов тестирования.

---

## Используемые технологии

* Java 11
* Maven
* JUnit
* Mockito
* JaCoCo

---

## Описание задания

В проекте необходимо протестировать классы `Feline`, `Cat` и `Lion`.

Требования к тестированию:

* Для каждого публичного метода каждого класса написан отдельный тест
* Для каждой ветки условий реализован отдельный тест
* В тестах используется параметризация там, где это оправдано
* Параметризованные тесты вынесены в отдельный класс
* Для зависимостей используются моки, созданные с помощью Mockito
* Класс `Lion` не зависит напрямую от класса `Feline`, используется инъекция зависимостей

---

## Архитектура проекта

* Проект собран с использованием Maven
* Тесты расположены в директории `src/test/java`
* Зависимости подключены только необходимые
* В проекте отсутствует лишний код и хардкод
* Код написан с учётом принципов модульного тестирования

---

## Покрытие кода

* Покрытие классов `Feline`, `Cat` и `Lion` составляет 100%
* Для анализа покрытия используется JaCoCo
